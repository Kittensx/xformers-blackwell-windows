# Changelog

## v0.1.0-blackwell-sm120

Initial experimental release.

- Added xFormers wheel validated on RTX 5070 Laptop GPU / SM120.
- Validated with PyTorch 2.11.0+cu128 and CUDA 12.8.
- Added companion MSLK dependency documentation.
- Added validation scripts for A1111 failure shape.
- Documented expected float16 success and float32 failure.
- Added SHA256 checksums.
